# Lula x Flávio Bolsonaro: Instagram Content Analysis (2026)

## Project Goal

This project analyzes Instagram feed posts published by Brazilian President **Luiz Inácio Lula da Silva (Lula)** and Senator **Flávio Bolsonaro** between **January 1 and July 5, 2026**.

The analysis aims to answer the following questions:

- What topics do Lula and Flávio Bolsonaro talk about most on Instagram?
- Who posts more frequently?
- Who receives more average likes per post?
- Who receives more average comments per post?

---

## Key Findings

- Flávio Bolsonaro's most common topic is **Lula, the Workers' Party (PT), or their supporters**.
- In contrast, Lula rarely mentions Flávio Bolsonaro. His posts focus mainly on **social policies**, including programs such as **Prouni**, **SUS**, and **Desenrola**.
- Flávio Bolsonaro posts more frequently than Lula and also receives more average likes and comments on Instagram feed posts during the period analyzed.

---

## Data Collection

The data were collected using **Apify**, based on the public Instagram profiles of Lula and Flávio Bolsonaro.

The scraper was configured to collect posts published between **January 1 and July 5, 2026**.

Although this filter was applied, Apify still downloaded some older posts from Flávio Bolsonaro's account. Therefore, an additional filtering step was performed in Python to ensure that only posts from the desired period were included in the analysis.

The maximum number of posts to download was set to **100,000** for each request to ensure that no posts were missed.

---

## Data Analysis

The analysis was conducted in **Python** using **Pandas** inside Jupyter Notebooks.

Separate notebooks were created for:

- Lula's posts
- Flávio Bolsonaro's posts
- A quantitative comparison between both accounts

### Data Cleaning

For both datasets, the following preprocessing steps were performed:

- Imported the required libraries (Pandas, Requests, OS, dotenv, etc.).
- Loaded the API key for OpenRouter using `python-dotenv`.
- Removed unnecessary columns generated by Apify.
- Converted the publication date column to Pandas datetime format.
- Created additional time variables (such as publication month) for temporal analysis.
- Removed posts without captions, since the analysis focused on caption content.

---

## AI-Assisted Topic Classification

To classify post topics, an AI-based workflow was developed.

The process was as follows:

1. A random sample of **20%** of the posts from each account was selected.
2. A `ground_truth` column was created, and every sampled post was manually categorized.
3. An OpenRouter language model was prompted to classify the same sample.
4. The AI predictions were compared with the manually assigned labels.
5. Classification accuracy was calculated.
6. The prompt was refined to improve performance.
7. After achieving an accuracy above **70%** for both datasets, the model was used to classify all remaining posts.
8. Finally, the frequency of each topic category was analyzed and compared between the two accounts.
